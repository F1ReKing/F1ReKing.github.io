Mac环境配置adb不起作用解决方案

发表于 2016-11-13 | 分类于 法.手段方法

在mac下配置好了adb之后，发现每次要使用的话 都需要source .bash_profile后才能使用。这是一个大坑啊，索性就研究下如何避免这个操作。

经过查找原因之后，发现其实这个是zsh的锅，所以就去改zsh的配置文件了。

解决方案：
在User目录在找到.zshrc文件，在文件的最后添加上

# Mac # Tools
设计模式之单例模式
iterm2设置代理
